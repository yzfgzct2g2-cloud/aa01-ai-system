# 我的AA01撰寫風格

1. 文字以口語化且正式為主。
2. 避免大量條列式拼接。
3. 優先描述個案實際生活情況。
4. 問題分析需與評估結果一致。
5. 不直接複製量表文字。
6. 目標需可執行且可追蹤。
7. 避免過度醫療化語句。
