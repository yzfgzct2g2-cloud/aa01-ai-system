# 退件紀錄格式

> 目前尚無實際退件案例。本檔僅定義紀錄格式，待真實退件發生後再依此格式累積。
> 嚴禁填入虛構退件原因或虛構照專偏好。

每筆退件以下列格式記錄（可一案多筆）：

```
## 退件紀錄 #編號

- 日期：YYYY-MM-DD
- 個案類型：（如：純居服 / 居服+交通 / 失智 / 獨居 …）
- 照專單位 / 代稱：
- CMS 等級：
- 服務碼涉及：（如 BA13、DA01 …）

### 退件原因（照專原文或重點）
1.
2.

### 問題分類
- [ ] 評估與服務不一致
- [ ] 目標不可執行 / 不可追蹤
- [ ] 服務量或頻率不合理
- [ ] 文字過於醫療化 / 過於條列
- [ ] 缺少來源依據
- [ ] 其他：

### 修正方向
-

### 修正前文字
> （貼上原文）

### 修正後文字
> （貼上修正後並通過之版本）

### 可重複使用的修正心得
- （此照專偏好 / 通用心得，供後續產文參考）
```
